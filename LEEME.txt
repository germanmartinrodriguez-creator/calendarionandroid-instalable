Instalación recomendada:
1. Crear un repositorio nuevo, por ejemplo calendario-barcelona-final.
2. Subir TODO el contenido de esta carpeta al repositorio.
3. Activar GitHub Pages: Settings > Pages > Deploy from branch > main > /root.
4. Abrir la URL nueva en Chrome Android.
5. Esperar unos segundos y usar Menú > Instalar app.

Si no aparece Instalar app:
- No uses la URL vieja.
- Borra iconos anteriores del calendario.
- En Chrome borra datos del sitio antiguo.
- Abre la URL nueva del repositorio nuevo.
